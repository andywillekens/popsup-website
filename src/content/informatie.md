# Informatie

### Hoe koppel ik PopsUp met Klaviyo?

Om een pop-up campagne te koppelen met Klaviyo heb je het volgende nodig:

- Een pop-up campagne
- Klaviyo account
- Een list ID van Klaviyo
- Een API key van Klaviyo
